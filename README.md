# nukleon-bot

Bot Discord fait avec discord.js v13 et TypeScript avec un dashboard.

Liste des catégories du bot:

* owner
* utils
* info
* music (bientot)
* nsfw (bientot)
* moderation (bientot)
* fun (bientot)
